# 🎫 Sipter-2: Sistema de Gestión de Tickets (Java MVC)

### 🚀 Descripción del Proyecto
Sipter-2 es una solución de software desarrollada en **Java** diseñada para optimizar la gestión y seguimiento de tickets en entornos operativos. Este proyecto aplica una arquitectura **MVC (Modelo-Vista-Controlador)** para garantizar un código escalable, organizado y fácil de mantener.

Este sistema nace de la necesidad de automatizar procesos administrativos, permitiendo un control riguroso sobre las solicitudes y estados de los servicios.

---

### 🛠️ Tecnologías Utilizadas
* **Lenguaje:** Java ☕
* **Arquitectura:** Modelo-Vista-Controlador (MVC)
* **Interfaz:** JavaFX / Swing (según implementación)
* **Gestión de Versiones:** Git & GitHub

---

### 📂 Estructura del Repositorio
Para facilitar la revisión técnica, el código se ha organizado de la siguiente manera:
* **`Sipter/src/Controlador`**: Lógica de negocio y manejo de eventos.
* **`Sipter/src/Modelo`**: Definición de objetos y acceso a datos.
* **`Sipter/src/vista`**: Componentes de la interfaz de usuario.
